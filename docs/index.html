ECTerm is designed to be a system-alternative terminal with a better feel to it. Originally, ECTerm was a personal project concept to replace the default windows terminal, but over some time and planning, it easily became less of a concept, and more of a goal.

ECTerm runs natively in your terminal, letting you have your original terminal open without the hassle of having a whole other GUI.


The Tango Desktop Project terminal icon.

Programmed in Rust, a programming language that ranked "most loved" for the fifth year in a row in Stack Overflow's Developer Survey, optimized for extra speed and stability, ECTerm will run fast, smooth, and free.

Wondering what Rust is? Check out the official site, or check out my explanation of it.
